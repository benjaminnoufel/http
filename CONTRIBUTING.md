# Contributing

## Requirements

- [Git][git]
- [Docker][docker]
- [Docker Compose][docker-compose]
- [GNU/Make][make]

## Installation

```console
$ git clone git@github.com:benjaminnoufel/http.git
```

## Install

```console
$ make install
```

## Lint

```console
$ make lint
```

## Test

```console
$ make test
```

## Build

```console
$ make build
```

## Types

```console
$ make types
```

## Clean

```console
$ make clean
```
